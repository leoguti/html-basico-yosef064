# HTML Básico - Escuela OSM + Vibe Coding

Bienvenido a tu primer proyecto de código. Aquí vas a aprender HTML desde cero.

**¿Qué es HTML?** Es el lenguaje que usan TODAS las páginas web del mundo. Cada vez que abres una página en tu navegador, estás viendo HTML.

**¿Ya lo has visto?** ¡Sí! El mapa del Salesiano (`index.html`) está hecho con HTML. Hoy vas a entender cómo funciona.

## ¿Cómo editar archivos aquí?

1. Haz clic en el archivo que quieres editar
2. Haz clic en el **lápiz** (ícono de editar) arriba a la derecha
3. Haz tus cambios
4. Escribe un mensaje de commit describiendo qué hiciste
5. Selecciona **"Commit directly to the main branch"**
6. Clic en **"Commit changes"**
